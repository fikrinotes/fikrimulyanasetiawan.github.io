Tentang Donkey.js:

Game ini adalah game yang serupa dengan game yang dibuat oleh Bill Gates ketika ia berumur 26 tahun, dengan judul donkey.bas . ".bas" sendiri adalah ekstensi untuk file yang dibuat dengan bahasa pemrograman BASIC. saya mencoba membuat game yang semirip mungkin dengan game buatan pendiri microsoft tersebut. Namun, saya menggunakan bahasa pemrograman javascript (vanilla javascript) untuk membuatnya. karena itu saya menamai game ini dengan nama Donkey.js.
